私人伺服器歷史


某位平凡到不行的小學生突然看到YouTube上有許多知名Minecraft實況主遊玩的影片，
發現居然是一個伺服器！而這位小學生原本就有開Minecraft伺服器的基礎，對他來說
根本是輕而易舉的事情，於是他便先邀請同班同學和親戚一起來玩，在2022年3月5日在
Aternos建立免費伺服器，這位小學生原本以為這樣就可以玩很久，然而在2022年3月11
日親戚突然說要不要用fabric的carpet模組開呢?就這樣經過許多測試成功轉移到fabric
伺服器，但是這時因為Aternos不支援carpet模組因此遷移到創辦的小學生電腦，從此開
始變成開關伺服器的工具人，就這樣大部分玩家發展到了全自動機械進度。
